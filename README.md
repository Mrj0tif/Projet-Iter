# Projet-Iter

Ce projet est issue de la participation a un concour (concour Iter).

Notre but étant de faire un robot trieur sur deux axe (X et Y), qui va s'occuper de reccupérer des cubes, et les trier par couleurs.


Le controleur utilisé est "carte mère" d'une ancienne imprimante 3D, la ultimaker v2.1.4.

Vous pouvez retrouver si dessus les pins qui ont été retrouver et qui sont mappé en fonction de leur appellation dans le programe.
